# Dev Journal 1.0.4 Release Notes

## Build targets
- Platform: macOS
- Architecture: Apple Silicon (arm64)
- Artifact: `Dev-Journal-1.0.4-macos-arm64.dmg`
- SHA-256: generated by `Build Desktop Release` workflow and attached to release asset `.sha256`

- Platform: Windows
- Architecture: x64
- Artifacts:
  - `Dev-Journal-1.0.4-windows-x64.msi`
  - `Dev-Journal-1.0.4-windows-x64-setup.exe`
- SHA-256: see `checksums-windows.txt` in GitHub Release assets.

## Included scope
- Planner expansion: Today Dashboard, Weekly Review, focus session tracking, daily wins, and tomorrow planning.
- Meetings domain: recurrence/reminders/participants/notes/decisions/action items with task materialization.
- Tasks/Goals/Pages workflow upgrades (drag-and-drop board, recurring tasks, subtasks, goal linking, gantt, and richer markdown pages).
- UI refresh with glassmorphism shell + updated app icon and layout fixes.

## Release checklist
1. Push `main` with version `1.0.4`.
2. Push git tag `v1.0.4`.
3. Verify workflow `Build Desktop Release` completed successfully.
4. Verify `.dmg`, `.msi`, `.exe`, `.sha256`, and `checksums-windows.txt` are attached to release `v1.0.4`.
